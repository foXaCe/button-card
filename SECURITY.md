# Security Policy

## Supported versions

Only the latest release published from the `main` branch is supported.
`@beta` prereleases (from the `dev` branch) are provided for testing and are not guaranteed to be supported.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report them privately via
[GitHub Security Advisories](https://github.com/foXaCe/button-card/security/advisories/new).

Target acknowledgement time: 7 days.
